### How to use BIOS ? 
<p>Place every BIOS at this location :</p>

> \.emulationstation\systems\retroarch\system

- 3DO => panafz10.bin
- Atari Lynx => lynxboot.img
- Game Boy Advance => gba_bios.bin
- Sony PlayStation => scph5500.bin / scph5501.bin / scph5502.bin
