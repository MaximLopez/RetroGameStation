==== English ====
Copy your Amiga CD32 ROMs here.
Supported extensions are: .uae .m3u .mds .chd .ccd .cue .nrg .iso

This system can be emulated using 3 emulators.
These emulators are listed below, in order of compatibility/ease of use:
amiberry-amiberry supports files of .cue .iso .ccd and is NOT Netplay compatible.
libretro-uae4arm supports files of .cue .ccd .chd .nrg .mds .iso .uae .m3u and is NOT Netplay compatible.
libretro-puae supports files of .cue .ccd .chd .nrg .mds .iso .uae .m3u and is NOT Netplay compatible.

Device requirements:
- A controller is required
- A keyboard is optional
- A mouse is optional
